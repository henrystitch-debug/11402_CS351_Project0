# Two Sum - Project 0

## Project Overview

This project implements a solution to the classic Two Sum problem, a fundamental algorithm challenge commonly used in coding interviews and competitive programming. The solution efficiently finds two distinct indices in an array whose elements sum to a target value.

## Learning Objectives

- Understand and implement array searching algorithms
- Explore time and space complexity trade-offs
- Practice problem-solving with hash maps/dictionaries
